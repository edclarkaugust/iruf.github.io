安装node-sass失败，解决办法

今天打包文件的时候，像往常一样输入 npm i ,出现了安装错误。往上翻报错发现是 node-sass 这个依赖安装失败。查了一下发现是被墙了，换了淘宝镜像和采取操作之后，还是安装出错。找到了解决办法，记录下来。如下：

node-sass 安装失败的原因

安装 node-sass 时在 node scripts/install 阶段会从 github.com 上下载一个 .node 文件，大部分安装不成功的原因都源自这里，因为 GitHub Releases 里的文件都托管在 s3.amazonaws.com 上面，而这个网址在国内总是网络不稳定

快速解决

选择和人生
